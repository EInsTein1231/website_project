Roadmapüî•

Level 1


HTML


When starting out to learn code, HTML, a website development language is by far the easiest programming language to start with to learn the basic concepts of structuring code, creating functions and connecting pages

Level 2


CSS


After mastering HTML, CSS will allow you to create the designs, colours, borders which gives structure and readability of your website's content

Level 3


JAVASCRIPT


Level 3 is Javascript, which is the most popular and widely used programming lanugage in almost every industry and workforce in the world. It is the language which creates the cool animations, interactive buttons that we use on our websites

Level 4


Python


Python is where we start to automate tasks, and conduct data analysis. This is applied in not only websites but other complex softwares which requires automation and data analysis because of its complexity.

Level 5


React JS


React JS is a Javascript library mainly used for building the UI or User Interface which is the face of the website, and after learning JavaScript, React JS will be much easier to learn.

Level 6


C++


C++ is the language which is used for game development, software engineering, developing operating systems and much more. Its a complex and difficult language to learn, but with experience of programming with Python and Javascript, C++ will be much easier to learn.

Level 7


SQL


SQL stands for Structured Query Language, which is used for managing data in a database management system for easy access and transfer of data within a system.
