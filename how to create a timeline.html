The front end web development focuses on the client side where the developers work with graphic designers or UI/UX designers to create the layout, content and experience of the website. The development team comprises the web content developer, server scriptor and the network security developer.


The difference between front end and back end developers


In the world of software engineering, there are those specialised at front end, back end and those that are good at both, known as full stack developers a.k.a "Superman".


But this brings up the questions, what is front end and back end? Firstly, web development is the process of building websites, which is different from UI UX which focuses more on the design, web development is on the website functions.


Front End Development


Front end developers program with the user in mind. Front end is a style of computer programming that focus on creating the elements and features of a website that will nbe seen by the user. You can also think of front Front end development is the programming which creates the features that is visible by the user. This is also known as the client side.


They will use knowledge about human psychology to create aesthetics and designs with colours to create high quality user experience. These can be translated through the elements such as buttons, navigation bar, layouts, graphics, images, animation and the structure of content.


Back End Development


Back end development focuses on the development on the site's performance and interactiveness, known as the server side. For example, a social media website has a lot of data, and to update and keep the data secured and easily accessible by the user, there has to be a place where its all accessible. This is called the database, or storage center and the back end developer is to code the connection that links the database with the website. Apart from that, they also work on troubleshooting, debugging and database management.


So in summary, the difference between front end dev and back end, is front end focuses on the visual elements such as the buttons, designs, colors and layout of the website, while the back end works on the performance and interactive aspects of the website. Together, they create a dynamic website.


Examples of dynamic websites:



1. Instagram: a social media website which relies on user generated content as the feed that the user sees.

2. YouTube: a video media website and application which relies on creators to create content to be on the viewers feed

3. Google: A search engine which uses user created websites and applications to be used by the public.

Once you have decided on your domain name, choose a domain name extension ( .com / .au / .edu / .gov ) depending on your company service, location and purpose.
